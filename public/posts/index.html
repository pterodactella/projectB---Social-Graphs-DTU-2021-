During the preliminary analysis performed during the first part of the project, the following was descovered: The total number of unique characters was found to be 298 (the dataset does not contain any duplicates) The total number of nodes is 255 with 17 isolated nodes The total number of links was found to be equal with 1196 The top connected character for the in-degrees is Michael Scott with a degree of 108 The top connected character for the out-degrees is Andy Bernard with a degree of 33 The memory usage for all the data is equal to 6.

Project A

This blog post focuses on providing the full analysis performed during the first part of the project. Gathering the Data To explore The Office, several datasets were used: First, all characters that played in the office from the first season to the final one were extracted using The Offica Wiki (Dunderpedia) A description of each character is downloaded from Dunderpedia. The data is used to extract the sentiment of each character and will be later compared to the sentiment extracted from the episodes transcript.
